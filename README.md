# 🌿 PlantCaare
### *Your Personal Botanical Sanctuary*

**PlantCaare** is a minimalist, full-stack web application designed for plant enthusiasts to track their greenery and never miss a watering day. Built with a focus on clean UI/UX, it demonstrates modern React patterns and real-time database integration.

---

## ✨ Key Features

* **Smart Dashboard:** A personalized overview of your collection with real-time stats (Total Plants, Member Status).
* **Dynamic Plant Cards:** Beautifully designed cards featuring:
    * **Status Badges:** Visual indicators for "Thirsty" vs. "Happy" plants.
    * **Actionable UI:** Log watering events and manage your collection with one click.
* **Automated Scheduling:** Automatically calculates next watering dates based on plant-specific frequency.
* **Minimalist Design:** A "nature-first" aesthetic using soft greens, glassmorphism, and responsive layouts.
* **Care Sanctuary:** A curated home page featuring natural imagery and daily botanical tips.

---

## 🚀 Tech Stack

* **Frontend:** React (Vite)
* **Database:** Firebase Firestore (NoSQL)
* **Routing:** React Router v6
